Components

We include several beautiful components that are easy to customize and implement.

Box

To create those beautiful boxes, you just need to add the .box class to the element.
For example, to create a contained box: 

<div class="container">
  <div class="row-fluid">
    <div class="span12 box">
        <!-- Your content here -->
     </div>
  </div>
</div>

If you want to create boxes inside boxes separated with a border botom, add the class .separator between rows:

<div class="container">
  <div class="row-fluid">
    <div class="span12 box">
      <div class="row-fluid">...</div>
      <div class="separator"></div>
      <div class="row-fluid">...</div>
    </div>
  </div>
</div>

Box Styling

You can add contrast between boxes, using .bg-dark class.

Padding

For easy customization we havent specified padding in the boxes. You add the padding. We include several padding classes:

Padding top: padding-top-n. ie. padding-top-50(Values range 0-100)

Padding right: padding-right-n. ie. padding-right-50(Values range 0-100)

Padding bottom: padding-bottom-n. ie. padding-bottom-50(Values range 0-100)

Padding left: padding-left-n. ie. padding-left-50(Values range 0-100)

Feel free to add you own classes.

Icons

To add an icon, you just need to add the class .entypo to the element and the icon's HTML code. For example:

<h4 class="entypo" style="font-size:50px">&#9874;</h4>

Check the character map to see the html code of each icon.

Social Icons

There's also a pack of social icons included, also from Entypo.

Example to include a social list:

Just add rel="lightwindow" to any a tag, and it will open a video or image in a lightwindow.

Misc

Here's some helper classes:

  • .align-left Aligns text or inline elements to left
  • .align-center Aligns text or inline elements to center
  • .align-right Aligns text or inline elements to right
  • .text-white Makes text and font icons white
  • .text-blackMakes text and font icons black
